The Network Planning Tool for Scotland is a web-based application designed to support strategic cycle network planning and investment decisions, primarily for use by local authorities.

This tool utilises various datasets, some of which have limitations. Users should always verify outputs against the most up to date geospatial data held by the local authority before proceeding with detailed design or investment decisions.

Please note that the commute layer results are based on 2011 Census data, and other trip purpose results are based on pre-pandemic data. These model-based estimates may not reflect current patterns which have changed since the pandemic.

Users are solely responsible for how they interpret and use the data. If you identify any errors or have concerns about the data or tool, please use the feedback form to report or provide feedback.
